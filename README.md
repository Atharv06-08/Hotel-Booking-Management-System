A console-based Hotel Room Booking System developed using C++ and Object-Oriented Programming (OOP). This project simulates real-world hotel operations with features like room management, customer tracking, automated billing, and persistent data storage.

🚀 Features
🛏️ Room Management
Tracks room types (Standard, Deluxe, Suite), pricing, and availability.
👤 Customer Management
Stores customer details and maintains stay history.
📋 Booking System
Handles check-in and check-out operations efficiently.
💰 Automated Billing
Calculates charges based on actual stay duration using system time.
🎯 Loyalty System
Customers are assigned tiers:
Normal
Silver (5% discount)
Gold (10% discount)
💾 Data Persistence
Uses file handling to save and load:
Rooms
Customers
Bookings
📊 Dashboard View
Displays:
Total rooms
Available rooms
Occupied rooms
Active bookings
🧾 Formatted Output UI
Clean, structured terminal interface for better readability.
🧠 Concepts Used
Object-Oriented Programming (Classes, Encapsulation)
File Handling (fstream)
STL (vector)
Time handling (ctime)
Input validation
Modular system design
